## Requirements

This project contains solutions to tasks solved during AI_devs 4.

### Configuration (public repo)

- Copy [`.env.example`](.env.example) to `.env` at the repo root. Set `AI_DEVS_HUB_BASE`, `AI_DEVS_API_KEY`, and every `AI_DEVS_HUB_PATH_*` (plus categorize response code vars if you run that solution) from **private** course materials — only path segments, never commit real base URLs, paths, or keys.
- Some solutions need extra local files (see `.env.example` and `*.example.json` under `solutions/`): e.g. `filtered_people.json`, `findhim_locations.json`, sendit mission file and `SENDIT_*` variables, drone `DRONE_*` prompts. **Do not publish** course-only task text or your own flags; strip them from git history before making the repo public again (e.g. BFG Repo-Cleaner).
